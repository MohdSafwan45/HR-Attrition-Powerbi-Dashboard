# HR Employee Attrition Analysis Dashboard

## Objective
Analysed IBM HR dataset of 1,470 employees to identify key drivers of employee attrition using Power BI and DAX.

## Tools Used
Power BI | Microsoft Excel | DAX | Power Query

## Dataset
IBM HR Analytics Dataset — 1,470 rows, 35 columns (Kaggle)

## Key Findings
- Employees working overtime had 3x higher attrition rate (30.5%) vs non-overtime (10.4%)
- Single employees showed highest attrition at 25.5% vs Married (12.5%)
- Sales Representatives had highest attrition (39.8%) among all 9 job roles
- Overall attrition rate: 16.1% (237 out of 1,470 employees)

## Dashboard Pages
1. HR Overview — KPI cards, department split, age group analysis
2. Attrition Analysis — Job role, overtime, marital status deep dive
3. Demographics & Salary — Salary bands, education, scatter plot
